# React + TypeScript + Vite

This pomodoro is a simple pomodoro timer built with React, TypeScript, and Vite.

It runs locally on your browser and is responsive.

## Features

- You can connect to your spotify account and play your favorite songs while working.
- You can set the duration of the pomodoro and the break time.
- You can select the sound you want to play when the timer ends.
- You can upload a sound file to play when the timer ends.

Sound format supported: mp3 and wav.

## Available Scripts

In the project directory, you can run:

### `npm dev`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `dist` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!
